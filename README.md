# SLEEP 2026 poster — references & contact

Landing page for the QR code on Connor Baggot's SLEEP 2026 poster, *Dim Light Melatonin Onset (DLMO) and Functional Connectivity in Later-Life Adults*. Live at <https://baggot4.github.io/sleep2026/>.

Contents: full reference list, contact + vCard, and a downloadable poster PDF.

## Rebuild

```bash
../.qr_venv/bin/python build_site.py
```

`build_site.py` regenerates `index.html`, `connor-baggot.vcf`, and the QR codes (`qr.svg`, `qr.png`, `qr-navy.svg`).
